Projet RMI

Objectif du projet

Le projet sur Java RMI a pour objectifs principaux de familiariser les étudiants avec la notion d’objet distribué, d’utiliser le passage de stub ou d’objet sérialisé, et de maîtriser le téléchargement de code via le codebase. En se plaçant dans le contexte d’un cabinet vétérinaire, les étudiants développeront une application permettant aux vétérinaires d’accéder et de gérer les dossiers des animaux à distance. Ce projet offre ainsi une opportunité d’appliquer des concepts théoriques à un cas pratique tout en renforçant leurs compétences en programmation distribuée.
