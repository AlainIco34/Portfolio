Recolleur de fichier CSV

Objectif du projet

Le but de ce projet consiste à rechercher tous les fichiers d’extension csv à partir du dossier dont le nom est donné en paramètre au script et recoller (sous certaines contraintes explicitées ci-après) les informations qui s’y trouvent dans un nouveau fichier csv dont le nom est également donné en paramètre au script. Les valeurs (hormis celles des premières lignes) des premières colonnes devraient correspondre à des noms de personnes mais les spécifications ne pourront le garantir. Le projet doit se faire en python et n’utiliser aucun module hormis os, re et sys. Dans le module os, la fonction walk() ne doit pas être utilisée.
