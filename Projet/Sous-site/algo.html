Implémentation et comparaison d’algorithmes

Objectif du projet

Ce projet porte sur l’implémentation de quatre algorithmes de recherche exacte de motifs : naïf, MP, KMP et Boyer-Moore. Les étudiants doivent programmer ces algorithmes en C++ (de préférence), C ou Java, dans un seul fichier bien commenté. Le programme doit lire le motif et le texte à partir de fichiers passés en ligne de commande, puis afficher le nombre de comparaisons effectuées pour chaque méthode, ainsi que les tables spécifiques (MP_next, KMP_next, Suff, D, R) pour les algorithmes concernés.

Une version améliorée de Boyer-Moore (BM2) avec une règle du mauvais caractère optimisée doit également être implémentée.
